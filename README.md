## Description
  <p align="center">It is a car rental backend service. It offers users the opportunity to rent a car between certain dates.</p>
    <p align="center">

## Dependencies
- TypeOrm
- Jwt 
- Postgresql
- Datefns
- Passport

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ nest start --watch

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
